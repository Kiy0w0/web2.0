# Website 2.0


## Features
- **Retro Music Player:** Windows Media Player / Winamp style UI.
- **Discord Status:** Real-time discord activity via Lanyard API.
- **Digital Clock:** WIB time display (UTC+7).
- **Navigation Drawer:** Collapsible sidebar menu.
- **Retro Widgets:** Dynamic calendar and animated header kaomojis.

## Tech
- Vanilla HTML, CSS, and JavaScript.
- Lanyard API for Discord integration.
- DSTN for dynamic user avatars.
